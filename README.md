# Ejercicios JavaScript

Colección de 9 ejercicios interactivos desarrollados con HTML, CSS y JavaScript puro (Vanilla JS), sin dependencias externas. Cada ejercicio demuestra conceptos fundamentales de la manipulación del DOM, manejo de eventos y lógica de programación en el navegador.

## 🌐 Demo en vivo

Accede al índice de ejercicios en GitHub Pages:
**[https://revertdeveloper.github.io/Ejercicio_Javascript/](https://revertdeveloper.github.io/Ejercicio_Javascript/)**

---

## 📋 Listado de ejercicios

| # | Nombre | Descripción |
|---|--------|-------------|
| 1 | [Cambiador de Color de Fondo](js_ejercicio_1.html) | Genera y aplica un color de fondo aleatorio en formato hexadecimal al hacer clic en un botón. |
| 2 | [Contador de Clics](js_ejercicio_2.html) | Lleva la cuenta del número de veces que el usuario hace clic sobre un botón. |
| 3 | [Lista Dinámica](js_ejercicio_3.html) | Permite añadir y eliminar elementos de una lista de forma dinámica sin recargar la página. |
| 4 | [Filtro de Búsqueda en Tiempo Real](js_ejercicio_4.html) | Filtra los elementos de una lista mientras el usuario escribe, ocultando los que no coinciden. |
| 5 | [Calculadora Sencilla](js_ejercicio_5.html) | Realiza las cuatro operaciones aritméticas básicas (suma, resta, multiplicación y división) con validación de entrada. |
| 6 | [Temporizador con Inicio, Pausa y Reinicio](js_ejercicio_6.html) | Cronómetro con controles de inicio, pausa y reinicio que muestra el tiempo en formato HH:MM:SS. |
| 7 | [Generador de Contraseñas Aleatorias](js_ejercicio_7.html) | Genera contraseñas seguras de longitud configurable usando letras, números y caracteres especiales. |
| 8 | [Contador de Palabras y Caracteres](js_ejercicio_8.html) | Analiza en tiempo real el texto introducido por el usuario y muestra el total de palabras y caracteres. |
| 9 | [Lista de Tareas con LocalStorage](js_ejercicio_9.html) | Gestiona una lista de tareas pendientes con persistencia de datos en el `localStorage` del navegador. |

---

## 🛠️ Tecnologías utilizadas

- **HTML5** — Estructura semántica de cada ejercicio.
- **CSS3** — Estilos y presentación visual.
- **JavaScript (ES6+)** — Lógica e interactividad sin librerías externas.

---

## 🚀 Ejecución local

No se requiere ninguna instalación ni servidor especial. Basta con clonar el repositorio y abrir el archivo `index.html` en cualquier navegador moderno:

```bash
git clone https://github.com/RevertDeveloper/Ejercicio_Javascript.git
cd Ejercicio_Javascript
# Abrir index.html en el navegador
```
